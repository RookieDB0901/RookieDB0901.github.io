C++ set容器

  • c++
  • notes
  • 容器

set/multiset属于关联式容器，所有元素都会在插入时自动被排序，底层结构是用二叉树实现。
set和multiset区别：

  • set不可以插入重复数据，而multiset可以
  • set插入数据的同时会返回插入结果，表示插入是否成功
  • multiset不会检测数据，因此可以插入重复数据
