C++ vector容器

  • c++
  • notes
  • vector
  • 容器

vector数据结构和数组非常相似，也称为单端数组，不同之处在于数组是静态空间，而vector可以动态扩展，即：

  • vector并不会在原空间之后续接新空间，而是找更大的内存空间，然后将原数据拷贝新空间，释放原空间
  • vector容器的迭代器是支持随机访问的迭代器
